**🔐 UPI QR Threat Analyzer**

A Python-based cybersecurity tool that scans QR codes and analyzes UPI payment data to detect potential threats and fraudulent patterns using heuristic analysis.

**📖 Overview**

UPI QR Threat Analyzer is designed to identify suspicious or potentially malicious QR-based payment requests. It extracts UPI data from QR codes and evaluates multiple risk factors such as unknown handles, missing transaction details, and scam-related keywords to generate a risk score.

---------------------------------------------------------------------------------------------

**Features**

  - 📷 QR Code Detection using OpenCV
  
  - 🔎 UPI Data Extraction (ID, Name, Amount, Note, etc.)
  
  - ⚠️ Heuristic Risk Analysis System
  
  - 🚨 Suspicious Keyword Detection
  
  - 📊 Risk Classification (Low / Medium / High)
  
  - 🖥️ GUI Interface using Tkinter
  
  - 📋 Detailed Analysis Report

---------------------------------------------------------------------------------------------

**🛠️ Tech Stack**
  
  - Python
  
  - OpenCV
  
  - Tkinter
  
  - Pillow
  
  - urllib

---------------------------------------------------------------------------------------------

**🚀 Installation**

    pip install opencv-python pillow

**▶️ Usage**
 
    python main.py
  
  - Enter the QR code image path
  
  - Click Start
  
  - View analysis report in GUI

---------------------------------------------------------------------------------------------

**Output**

**The tool provides:**
  
  - UPI ID and Name
  
  - Risk Score
  
  - Risk Level (Low / Medium / High)
  
  - List of detected risk factors

---------------------------------------------------------------------------------------------

**🎯 Use Case**

  - Detect potential UPI QR scams
  
  - Analyze unknown payment requests
  
  - Educational tool for cybersecurity beginners
  
  - Demonstration of basic threat analysis techniques
